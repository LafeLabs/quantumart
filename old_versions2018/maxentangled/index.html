Maximally entangled two qubit states with HTML and CSS representation.

$$\alpha\left|0\right> + \beta\left|1\right>$$ $$\cos{\left(\frac{\theta}{2}\right)}\left|0\right> + \sin{\left(\frac{\theta}{2}\right)}e^{i\phi}\left|1\right>$$ Maximal entanglement is when $$|\alpha|^2 = |\beta|^2 = \frac{1}{2}$$ or $$|\alpha| = |\beta| = \frac{1}{\sqrt{2}}.$$ Thus maximal entanglement is when $\theta = \frac{\pi}{2}$, with any value of $\phi$, making this the equator of the Bloch sphere as expected. In the CSS faded and rotating cat picture representation, this is a pair of cats both with transparrency of 0.5, one of which is the ground state and does not roteate, and the other freely rotating to all angles to represent the phase angle.

$\frac{1}{\sqrt{2}}\left|0\right>$ $\frac{1}{\sqrt{2}}e^{i\phi}\left|1\right>$

For 2 qubits, the states are $$\alpha\left|00\right> + \beta\left|01\right> + \gamma\left|10\right> + \delta\left|11\right>, $$ which can be parameterized as For maximally entangled states $$|\alpha|^2 = |\beta|^2 = |\delta|^2 =|\gamma|^2 = \frac{1}{4}$$ or $$|\alpha| = |\beta|= |\delta| =|\gamma| = \frac{1}{2}.$$

While the above algebra is not strictly needed to express the maximally entangled states in CSS it will be needed for the JavaScript control of them in a logical way and also for the 3d representations to come. The set of maximally englangled states for the cat pictures with CSS are all four cats having a opacity of 0.25, the ground state cat (Grumpy Cat) having zero rotation, and any arbitrary rotation for the other three cats (Keyboard Cat, Lil Bub, and Maru)

Tap on cats' faces to rotate(only works for states other than ground state):

$\frac{1}{2}\left|00\right>$ $\frac{1}{2}e^{i\phi_0}\left|01\right>$ $\frac{1}{2}e^{i\phi_1}\left|10\right>$ $\frac{1}{2}e^{i\phi_2}\left|11\right>$

These complex amplitudes can be parameterized as follows: $$\cos{\left(\theta_0\right)}\left|00\right> + \cos{\left(\theta_1\right)}\sin{\left(\theta_0\right)}e^{i\phi_0}\left|01\right> + \cos{\left(\theta_2\right)}\sin{\left(\theta_1\right)}\sin{\left(\theta_0\right)}e^{i\phi_1}\left|10\right> + \sin{\left(\theta_2\right)}\sin{\left(\theta_1\right)}\sin{\left(\theta_0\right)}e^{i\phi_2}\left|11\right>. $$

For the maximally entanlged states we can then compute the values of $\theta_0$, $\theta_1$ and $\theta_2$: $$\theta_0 = \arccos{\left(\frac{1}{2}\right)} = 60^{\circ} = \frac{\pi}{3}$$ $$\sin{\theta_0} = \sin{\left(\frac{\pi}{3}\right)} = \frac{\sqrt{3}}{2}$$ therefore $$\frac{\sqrt{3}}{2}\cos{\left(\theta_1\right)} = \frac{1}{2}$$ or $$\theta_1 = \arccos{\frac{1}{\sqrt{3}}} = \theta_m,$$ where $\theta_m\approx 54.7^{\circ}$ is the magic angle

The final angle is found from the fact that the last two equations imply that $$\sin{\theta_2} = \cos{\theta_2},$$ which implies that $\theta_2 = 45^{\circ} = \frac{\pi}{4}$

Now, in order to make simple geometry for a 3d model, I'll renormalize the angles so that $\theta_i=\frac{\pi}{2}$ is always the value for the maximally entangled state and rewrite the equations as follows: $$\cos{\left(\frac{2\theta_0}{3}\right)}\left|00\right> + \cos{\left(\frac{2\theta_m\theta_1}{\pi}\right)}\sin{\left(\frac{2\theta_0}{3}\right)}e^{i\phi_0}\left|01\right> + \cos{\left(\frac{\theta_2}{2}\right)}\sin{\left(\frac{2\theta_m\theta_1}{\pi}\right)}\sin{\left(\frac{2\theta_0}{3}\right)}e^{i\phi_1}\left|10\right> + \sin{\left(\frac{\theta_2}{2}\right)}\sin{\left(\frac{2\theta_m\theta_1}{\pi}\right)}\sin{\left(\frac{2\theta_0}{3}\right)}e^{i\phi_2}\left|11\right>. $$
